Начало работы
Откройте приложение в браузере

Используйте верхнее меню для навигации

Выберите нужный раздел из выпадающего списка

Поиск данных
Выберите тип поиска из меню (например, "Поиск по имени клиента")

Введите запрос в появившееся поле поиска

Нажмите кнопку "Найти" для выполнения поиска

Работа с таблицами
Сортировка: кликните на заголовок колонки для сортировки

Поиск: используйте соответствующие разделы меню для фильтрации данных

Создание сделок
Перейдите в раздел "Новая сделка"

Выберите опцию:

"Создать новую сделку" - ручной ввод

"Создать шаблон сделки" - создание многоразового шаблона

"Создать сделку по шаблону" - использование существующего шаблона

Структура данных
Основные сущности
Сделки (dealsData): название, клиент, сумма, дата, статус

Товары (stockData): название, категория, количество, цена

Клиенты (clientsData): имя, компания, телефон, регион

Шаблоны (templatesData): название, описание, список документов

Техническая информация
Используемые технологии
HTML5 - структура приложения

CSS3 - стилизация и адаптивный дизайн

JavaScript (ES6) - логика приложения

Google Fonts (Inter) - шрифты интерфейса

Архитектура
Монолитная структура (все в одном HTML-файле)

Клиентская обработка данных

Локальное хранение демо-данных

Модульная организация JavaScript кода
